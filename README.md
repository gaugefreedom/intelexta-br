# Intelexta Brasil

Site oficial da Intelexta em português do Brasil.

Este repositório reúne a presença pública da Intelexta para o mercado brasileiro, com foco inicial em:

- transparência no uso de IA
- integridade acadêmica e científica
- registro verificável do processo
- relatórios de integridade
- comprovantes verificáveis para pesquisa, ensino e revisão

## Objetivo

A primeira etapa deste projeto é lançar a homepage de `intelexta.com.br` em pt-BR.

Depois, o mesmo repositório poderá hospedar:

- `/blog` — artigos e notas em português
- `/docs` — documentação da API, SDK, schema e protocolo
- `/universidades` — página dedicada ao contexto acadêmico brasileiro

## Posicionamento

A Intelexta não é um detector de IA.

A Intelexta adiciona uma camada de verificação ao trabalho assistido por IA, transformando rascunhos, análises e documentos em artefatos mais legíveis, portáveis e verificáveis.

Em vez de perguntar apenas “você usou IA?”, a proposta é permitir algo mais forte:

**mostre o processo, os checks e o comprovante.**
